# Manikanta's Professional Portfolio

A modern, responsive portfolio website built with HTML5, CSS3, and JavaScript featuring stunning UI/UX effects and animations.

## ✨ Features

### Design & UI/UX
- **Gradient Backgrounds**: Beautiful gradient overlays and animated backgrounds
- **Smooth Animations**: Fade-in, slide-in, and parallax effects
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Interactive Elements**: Hover effects, button animations, and ripple effects
- **Glassmorphism**: Modern glass effect on cards and elements
- **Custom Cursor**: Enhanced visual feedback
- **Twinkling Stars**: Animated star field in hero section

### Sections
1. **Navigation Bar**: Fixed navbar with smooth scrolling
2. **Hero Section**: Eye-catching landing section with animations
3. **About Me**: Personal introduction with statistics
4. **Skills**: Grid layout showcasing technical skills
5. **Projects**: Featured projects with descriptions and links
6. **Contact**: Contact information and message form
7. **Social Links**: Links to social media profiles
8. **Footer**: Copyright and credits

### Technical Highlights
- Smooth scroll behavior
- Intersection Observer API for element animations
- CSS Grid and Flexbox layouts
- CSS animations and transitions
- Responsive media queries
- Form validation and submission handling
- Mobile-responsive navigation

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Internet connection (for Font Awesome icons)

### Installation
1. Clone the repository:
```bash
git clone https://github.com/manikanta2208/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Open `index.html` in your browser or deploy to GitHub Pages

## 🎨 Customization

### Update Personal Information
Edit the following in `index.html`:
- Name and subtitle in hero section
- About section text
- Project details
- Contact information
- Social media links

### Change Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #8b5cf6;
    --accent: #ec4899;
    --dark: #0f172a;
    --darker: #020617;
    --light: #f8fafc;
    --text: #64748b;
    --border: #e2e8f0;
}
```

### Add New Projects
```html
<div class="project-card">
    <div class="project-image">
        <div class="image-placeholder-project">
            <i class="fas fa-project-diagram"></i>
        </div>
    </div>
    <div class="project-info">
        <h3>Project Title</h3>
        <p>Project description</p>
        <div class="project-tags">
            <span>Technology</span>
        </div>
        <div class="project-links">
            <a href="#" class="link-btn">View Project</a>
            <a href="#" class="link-btn">GitHub</a>
        </div>
    </div>
</div>
```

## 📦 Deployment

### GitHub Pages
1. Go to Settings > Pages
2. Select `main` branch as source
3. Your portfolio will be live at `https://manikanta2208.github.io/portfolio`

### Netlify
1. Connect GitHub repository
2. Deploy directly
3. Your portfolio is live!

### Vercel
1. Import project from GitHub
2. Click Deploy
3. Your portfolio is live!
